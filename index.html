Manajemen Tahfidh Juz Amma & Absensi Siswa

Versi 2.3.5
Fitur Aplikasi
  • 📖
    Tahfidh Juz 30 Input setoran harian per siswa dengan pilihan surat Juz 30, rentang ayat, status Lancar / Kurang Lancar, dan keterangan. Riwayat setoran per tanggal bisa dilihat, diedit, dan diekspor ke Excel.
  • 🎯
    Pencapaian Awal Siswa Catat posisi hafalan awal siswa (surat & ayat terakhir) secara otomatis maupun manual per surat, lengkap dengan keterangan Belum Khatam / Muroja'ah / Tasmi'.
  • 👥
    Absensi Terintegrasi Import siswa sekali untuk Tahfidh dan Absen. Tandai Hadir / Ijin / Sakit / Alpha per sesi, lihat riwayat per tanggal, dan ekspor laporan per kelas ke Excel.
  • 📊
    Statistik & Rekap Tab Statistik menampilkan grafik performa mingguan, leaderboard kelas, rekap status hafalan (Belum Khatam / Muroja'ah / Tasmi') dengan filter per kelas, serta trend 30 hari dan insight AI.
  • 🏅
    Gamifikasi & Badge Streak harian, badge pencapaian otomatis, dan progress ring hafalan untuk memotivasi siswa terus setoran.
  • 📂
    Viewer File Excel Upload file setoran atau absensi (.xlsx/.xls/.csv) untuk ditampilkan, difilter per status dan kelas, tanpa perlu buka aplikasi lain.
  • 💾
    Firebase & Offline / PWA Data tersinkron ke Firebase Firestore secara real-time dan tersedia offline. Dapat dipasang ke layar utama HP seperti aplikasi native.
